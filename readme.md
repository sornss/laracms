## About LaraCMS

Laravel and vue js based cms starter project for blogs

## Tech Stack
- Laravel 5.4
- Vue 2.0
- Mysql / MariaDb

## Requirement
- PHP >= 5.6.4
- OpenSSL PHP Extension
- PDO PHP Extension
- Mbstring PHP Extension
- Tokenizer PHP Extension
- XML PHP Extension

## Install Project
<code>
	composer install
</code>

## Database Migrations
<code>
	php artisan migrate
</code>

## Install node modules
<code>
	npm install
</code>
